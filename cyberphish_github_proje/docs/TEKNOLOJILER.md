# Kullanılan Teknolojiler

## Programlama

### Python 3
- **http.server**: Yerleşik HTTP sunucusu modülü
- **socket**: Ağ iletişimi ve IP tespiti
- **threading**: Çoklu işlem desteği
- **json**: Veri kaydetme ve raporlama
- **subprocess**: Sistem komutları çalıştırma
